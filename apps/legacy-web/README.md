# Legacy Web App

This package contains the original React Router/Vite frontend moved from `apps/web/src-legacy`.

## Run

```bash
pnpm --filter @digital-portfolio/legacy-web legacy:dev
```

## Build

```bash
pnpm --filter @digital-portfolio/legacy-web legacy:build
```

## Environment

Copy `.env.example` to `.env` and set API endpoints for your server.
